# git 基本常用命令
* 用git config配置 Git，要做的第一件事就是设置名字和邮箱地址：这个设置只是一个账户的标记
  * git config --global user.name "xxx" 设置你的名字
  * git config --global user.email "xxx" 设置你的email
* `git init` 在一个文件夹的`.git`文件夹下初始化一个仓库
* `git status` 查看当前仓库的状态
* `git add file.txt` 添加file.txt到待提交区 `add -A`添加当前文件夹里的所有修改内容
* `git commit -m "xxx"` xxx表示提交信息 如果没有后边的信息，git会进入vim让你补充提交的信息
* `git diff` 显示被跟踪的文件的修改状态
* `git log` 查看提交日志，历史
* `git merge` 
* `git pull`
* `git push`
* `git remote`
* `git fetch`
* `git checkout`
* `git branch`
